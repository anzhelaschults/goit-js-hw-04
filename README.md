# goit-js-hw-04

Homework assignment for GoIT JavaScript course. Topic 6: Arrays and Object Methods.
Домашнє завдання курсу JavaScript від GoIT. Тема 6: Масиви та методи об'єкта.

## Tasks / Завдання

- **Task 1** — `isEnoughCapacity(products, containerSize)`: checks if total quantity of all products fits in a container / перевіряє чи поміщається загальна кількість товарів у контейнер
- **Task 2** — `calcAverageCalories(days)`: calculates average daily calories consumed by an athlete during the week / обчислює середньодобову кількість калорій спортсмена за тиждень
- **Task 3** — `profile` object with methods `getInfo()`, `changeUsername(newName)`, `updatePlayTime(hours)` to manage a gaming profile / об'єкт профілю гравця з методами для управління іменем та ігровим часом
